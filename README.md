Scripts
-------------

To run script just type
```
yarn run <scriptName>
```
or
```
npm run <scriptName>
```

Here is a list of available scripts

Command | Notes
------- | -----
`start` | Runs dev enviroment
`build` | Create build version of you code in `/build` directory
`test` | Runs tests in `develop` mode and enable watcher
